# Debt Tracker - Premium Frontend

A high-fidelity web interface for tracking personal debts, featuring glassmorphism and real-time filtering. Perfect for deployment on **AWS Amplify**.

## 🚀 Deployment (AWS Amplify)
1.  Drag & Drop the files in this folder to a new Amplify App.
2.  Open your site URL.
3.  Enter your **API Gateway Invoke URL** in the "Connect API" field.
4.  *Optional*: Click the **Save** icon to persist the URL in `localStorage`.

## ✨ Key Features
- **Modern UI**: Built with HSL color palettes and backdrop-blur effects.
- **Smart Filter**: Search for a name to see their **entire list of records** and total debt instantly.
- **Dashboard Stats**: Real-time counter for active debts and settled payments.
- **Actionable List**: Mark any debt as **"Paid"** with a single click.

## 📦 Tech Stack
- HTML5 / CSS3 (Vanilla)
- JavaScript (Fetch API)
- Lucide Icons
- Google Fonts (Outfit)
